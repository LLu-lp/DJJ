# 佳悦影院管理系统 - 项目运行指南

## 一、项目概述

佳悦影院管理系统是一个完整的影院票务管理平台，包含用户购票、员工管理、管理员后台等功能模块。系统采用前后端分离架构，后端基于 Spring Boot + MyBatis Plus，前端基于 Vue 3 + Element Plus。

**项目结构：**
```
DJJ/
├── backend/          # 后端项目（Java + Spring Boot）
│   ├── src/
│   │   ├── main/
│   │   │   ├── java/com/jiayue/cinema/
│   │   │   │   ├── controller/      # 控制层
│   │   │   │   ├── service/         # 业务层
│   │   │   │   ├── mapper/          # 数据访问层
│   │   │   │   ├── entity/          # 实体类
│   │   │   │   ├── dto/             # 数据传输对象
│   │   │   │   ├── config/          # 配置类
│   │   │   │   └── exception/       # 异常处理
│   │   │   └── resources/
│   │   │       ├── application.yml  # 应用配置
│   │   │       └── db/
│   │   │           ├── init.sql     # 数据库初始化脚本
│   │   │           └── test_data.sql # 测试数据
│   │   └── test/
│   └── pom.xml                      # Maven 依赖配置
└── web/              # 前端项目（Vue 3）
    ├── src/
    │   ├── api/                     # API 接口调用
    │   ├── views/                   # 页面组件
    │   │   ├── admin/               # 管理员页面
    │   │   ├── staff/               # 员工页面
    │   │   ├── user/                # 用户页面
    │   │   └── auth/                # 认证页面
    │   ├── layouts/                 # 布局组件
    │   ├── router/                  # 路由配置
    │   ├── stores/                  # 状态管理
    │   └── utils/                   # 工具函数
    ├── package.json                 # npm 依赖配置
    └── vite.config.js               # Vite 构建配置
```

## 三、数据库配置

### 1. 使用 Navicat 导入数据库

**步骤一：打开 Navicat**
- 启动 Navicat 应用程序
- 点击左上角 "连接" 按钮，选择 "MySQL"
- 输入连接信息：
  - 连接名：jiayue_cinema
  - 主机名：localhost
  - 端口：3306
  - 用户名：root
  - 密码：你的 MySQL 密码
- 点击 "确定" 建立连接

**步骤二：创建数据库**
- 在左侧连接列表中右键点击 "jiayue_cinema" 连接
- 选择 "新建数据库"
- 数据库名称：jiayue_cinema
- 字符集：utf8mb4
- 排序规则：utf8mb4_unicode_ci
- 点击 "创建"

**步骤三：导入 SQL 文件**
- 在左侧选中 "jiayue_cinema" 数据库
- 点击菜单栏 "文件" → "运行 SQL 文件"
- 选择项目中的 `jiayue_cinema.sql` 文件
- 点击 "打开"，等待导入完成

**步骤四：验证导入**
- 在左侧展开 "jiayue_cinema" 数据库
- 检查是否能看到所有表（users、movies、orders 等）
- 右键点击表 → "查看数据" 验证数据是否正确导入

### 2. 修改数据库连接配置

编辑 `backend/src/main/resources/application.yml`：

```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/jiayue_cinema?useSSL=false&serverTimezone=UTC&allowPublicKeyRetrieval=true
    username: root              # 修改为你的 MySQL 用户名
    password: your_password     # 修改为你的 MySQL 密码
    driver-class-name: com.mysql.cj.jdbc.Driver
```

> **注意**: 确保 `url`、`username`、`password` 与你的 MySQL 连接信息一致

## 四、Maven 安装与配置

### 1. 下载 Maven

访问 [Maven 官方网站](https://maven.apache.org/download.cgi) 下载最新版本（3.8.1 或以上）

**下载步骤：**
- 点击 "Binary zip archive" 下载 `apache-maven-3.x.x-bin.zip`
- 解压到本地目录，例如：`C:\Program Files\apache-maven-3.9.0`

### 2. 配置环境变量

**Windows 系统：**
1. 右键点击 "此电脑" → "属性" → "高级系统设置"
2. 点击 "环境变量" 按钮
3. 在 "系统变量" 中点击 "新建"
   - 变量名：`MAVEN_HOME`
   - 变量值：`C:\Program Files\apache-maven-3.9.0`（你的 Maven 安装路径）
4. 编辑 "Path" 变量，添加：`%MAVEN_HOME%\bin`
5. 点击 "确定" 保存

**Mac/Linux 系统：**
```bash
# 编辑 ~/.bash_profile 或 ~/.zshrc
export MAVEN_HOME=/path/to/apache-maven-3.9.0
export PATH=$MAVEN_HOME/bin:$PATH

# 使配置生效
source ~/.bash_profile
```

### 3. 验证 Maven 安装

打开命令行/终端，运行：
```bash
mvn -version
```

如果显示 Maven 版本信息，说明安装成功。

### 4. 配置 Maven 镜像（可选，加快下载速度）

编辑 `MAVEN_HOME/conf/settings.xml`，在 `<mirrors>` 标签中添加：

```xml
<mirror>
    <id>aliyun</id>
    <mirrorOf>central</mirrorOf>
    <name>Aliyun Maven Mirror</name>
    <url>https://maven.aliyun.com/repository/public</url>
</mirror>
```

## 八、常见问题

### Q1: Maven 命令找不到

**解决方案：**
1. 检查 Maven 是否正确安装
2. 验证环境变量 `MAVEN_HOME` 是否设置正确
3. 重启命令行/终端，使环境变量生效
4. 运行 `mvn -version` 验证

### Q2: Maven 依赖下载缓慢

**解决方案：**
1. 配置 Maven 镜像（参考第四章节）
2. 使用阿里云镜像加速下载
3. 检查网络连接是否正常

### Q3: 后端启动失败，提示数据库连接错误

**解决方案：**
1. 检查 MySQL 是否正常运行
2. 验证 `application.yml` 中的数据库连接信息
3. 确保数据库 `jiayue_cinema` 已创建并导入了 SQL 文件
4. 检查 MySQL 用户权限

### Q4: 前端无法连接到后端

**解决方案：**
1. 确保后端已启动在 `http://localhost:8087/api`
2. 检查 `web/src/utils/request.js` 中的 API 基础 URL 配置
3. 检查浏览器控制台是否有 CORS 错误
4. 确保防火墙未阻止 8087 端口

### Q5: 前端页面显示空白

**解决方案：**
1. 清除浏览器缓存（Ctrl+Shift+Delete）
2. 检查浏览器控制台是否有 JavaScript 错误
3. 重新启动前端开发服务器
4. 确保 Node.js 版本 >= 16

### Q6: 登录后页面无法加载

**解决方案：**
1. 检查后端是否正常运行
2. 验证 JWT Token 是否正确生成
3. 检查浏览器 LocalStorage 中是否存储了 Token
4. 查看浏览器控制台的网络请求是否返回 401 错误

## 九、项目配置详解

### 后端配置文件 (application.yml)

```yaml
server:
  port: 8087
  servlet:
    context-path: /api

spring:
  application:
    name: cinema
  
  # 数据库配置
  datasource:
    url: jdbc:mysql://localhost:3306/jiayue_cinema?useSSL=false&serverTimezone=UTC
    username: root
    password: root
    driver-class-name: com.mysql.cj.jdbc.Driver
  
  # JPA 配置
  jpa:
    hibernate:
      ddl-auto: validate
    show-sql: false
  
  # MyBatis Plus 配置
  mybatis-plus:
    mapper-locations: classpath*:/mapper/**/*.xml
    type-aliases-package: com.jiayue.cinema.entity
    configuration:
      log-impl: org.apache.ibatis.logging.stdout.StdOutImpl

# JWT 配置
jwt:
  secret: your-secret-key-change-in-production
  expiration: 86400000  # 24 小时

# 日志配置
logging:
  level:
    root: INFO
    com.jiayue.cinema: DEBUG
```

### 前端配置文件 (vite.config.js)

```javascript
import { defineConfig } from 'vite'
import vue from '@vitejs/plugin-vue'
import path from 'path'

export default defineConfig({
  plugins: [vue()],
  resolve: {
    alias: {
      '@': path.resolve(__dirname, './src'),

    port: 5173,

      '/api': {
```

1. **编译打包**
```bash
cd backend
mvn clean package -DskipTests
```

2. **上传到服务器**
```bash
scp target/cinema-0.0.1-SNAPSHOT.jar user@server:/path/to/app/
```

3. **启动应用**
```bash
java -jar cinema-0.0.1-SNAPSHOT.jar --spring.profiles.active=prod
```

### 前端部署

1. **构建生产版本**
```bash
cd web
npm run build
```

2. **上传到服务器**
```bash
scp -r dist/* user@server:/path/to/nginx/html/
```

3. **配置 Nginx**
```nginx
server {
    listen 80;
    server_name your-domain.com;
    
    location / {
        root /path/to/nginx/html;
        try_files $uri $uri/ /index.html;
    }
    
    location /api {
        proxy_pass http://backend-server:8087/api;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
```

## 十四、管理员操作指南

### 电影和影院绑定流程

**问题**: 管理员新增电影后，如何将电影和影院绑定？

**解决方案**: 电影和影院的绑定是通过"排片"（Schedule）来实现的。具体步骤如下：

#### 第一步：新增电影

1. 登录管理员账号
2. 进入 **电影管理** 页面
3. 点击 **新增电影** 按钮
4. 填写电影信息：
   - 电影名称（必填）
   - 海报 URL
   - 类型（可多选）
   - 导演
   - 主演
   - 时长（分钟）
   - 上映日期
   - 状态（正在热映/即将上映）
   - 简介
5. 点击 **确认保存** 完成电影创建

#### 第二步：为电影创建排期（绑定影院）

1. 进入 **排片管理** 页面
2. 点击 **新增排片** 按钮
3. 填写排片信息：
   - **电影**（必填）：选择刚才新增的电影
   - **影院**（必填）：选择要放映的影院
   - **影厅**（必填）：选择影院中的具体影厅
   - **放映日期**（必填）：选择放映日期
   - **开始时间**（必填）：选择放映开始时间
   - **票价**（必填）：设置普通票价格
   - **VIP票价**（可选）：设置 VIP 票价格
   - **语言版本**：选择语言（国语/粤语/原声）
4. 点击 **确定** 完成排期创建

**说明**：
- 一部电影可以在多个影院创建排期
- 一个影院可以在不同时间放映多部电影
- 创建排期后，系统会自动初始化该场次的所有座位状态
- 用户就可以在前端选择该电影和影院进行购票

#### 示例流程

假设要在"佳悦影城(万达广场店)"放映电影"流浪地球2"：

1. **电影管理** → 新增电影 → 填写"流浪地球2"的信息 → 保存
2. **排片管理** → 新增排片 → 选择"流浪地球2" → 选择"佳悦影城(万达广场店)" → 选择"1号厅" → 选择日期和时间 → 设置票价 → 保存

完成后，用户就可以在前端看到该电影在该影院的放映场次，并进行购票。
