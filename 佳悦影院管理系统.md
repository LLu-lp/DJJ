# 佳悦影院管理系统

## 一、技术栈

|技术维度|核心技术/框架|说明|
|---|---|---|
|后端核心|Java + Spring Framework|包含Spring Core、Spring MVC，建议扩展使用Spring Boot（简化配置）、MyBatis/MyBatis-Plus（数据持久化）|
|前端核心|Vue3|配套核心库：Vue Router（路由）、Pinia（状态管理）、Axios（前后端交互）、Element Plus（UI组件库）|
|数据库|MySQL|存储用户、电影、订单、影院等核心数据|
|辅助技术|Redis|缓存热门电影、用户登录态、场次座位状态（提升性能）|
|辅助技术|JWT|实现用户/员工/管理员的身份认证与权限控制|
|辅助技术|ECharts|实现票房折线图、电影类型饼状图、票房柱状图等数据可视化|
|辅助技术|WebSocket|实现在线客服实时聊天功能|
